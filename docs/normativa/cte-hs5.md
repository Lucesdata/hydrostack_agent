## Volumen mínimo fosa séptica
V (m³) = (N × Q × T_r / 1000) + (N × 0.04 × T_limp)
- N = número de habitantes
- Q = dotación (L/hab·día)
- T_r = tiempo de retención (días, mín. 3)
- T_limp = período entre limpiezas (años, mín. 2)
- V_mínimo = 3 m³
